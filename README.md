# Ecommerce Sales Analysis

## Project Overview
This project analyzes ecommerce sales data using Python, SQL, and Tableau.

The project includes:
- Data cleaning using Pandas
- Exploratory Data Analysis (EDA)
- SQL queries for business insights
- Interactive Tableau dashboard

---

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- SQLite
- Tableau
- GitHub

---

## Dataset
Sample Superstore Dataset

---

## Key Insights
- West region generated highest profit
- Technology category had highest sales
- Monthly sales trends were analyzed
- Regional performance comparison was performed

---

## Files Included

### notebooks/
- Analysis.ipynb → Data cleaning and analysis notebook

- cleaned_superstore.csv → Cleaned dataset

### sql/
